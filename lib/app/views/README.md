```
views

  |     |__ settings
  |            |__ settings_page.dart
  |            |__ account_page.dart
  |            |__ language_page.dart
  |            |__ theme_page.dart
  |
  |__ screens
  |      |__ settings
  |             |__ settings_screen.dart
  |             |__ account_screen.dart
  |             |__ language_screen.dart
  |             |__ theme_screen.dart
  |
  |__ widgets
         |__ settings
                |__ account
                        |__ avatar_widget.dart
                        |__ wrapper_widget.dart
```
